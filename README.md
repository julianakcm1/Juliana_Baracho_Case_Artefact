**Date:** June 2026


## 📌 Overview

This repository contains my complete analytical solution for the Artefact selection process case.

The challenge was to design an analytical solution for a national e-commerce company facing four critical problems:

- Lack of trust in data between Marketing, Product, and Finance.
- Difficulty identifying which marketing channel actually scales the business.
- Customer loss is only noticed when revenue drops.
- Over one-third of orders are cancelled or returned.


## 🛠️ Tools & Technologies

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- RFM Model for customer segmentation
- Last-Touch attribution model
- Executive Report (PDF)


🔍 Key Findings

Based on 99,751 valid orders:

- **Revenue Leakage:** 33.23% of gross revenue lost to cancellations and returns.
- **Best Channels (ROI):** Email (+3.37%) and Newsletter (+2.24%).
- **Worst Channel (ROI):** GoogleAds (-32.73%) – losing money on every real spent.
- **Customer Health:**
  - 682 Champions (13.6%)
  - 1,699 At Risk (34.0%)
  - 1,602 Churn (32.1%)
